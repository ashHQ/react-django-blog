# React + Django Blog App

My first full stack developement

A full-stack blog app built using React (frontend) and Django REST Framework (backend). Built from scratch using **only official documentation** to improve dev skills.

## Features

- User registration & login
- Create/edit/delete posts
- Comment system
- Token-based authentication
- Responsive UI

## Tech Stack

- Backend: Django + DRF
- Frontend: React + React Router
- Auth: Token (DRF)
